## Sobre mí

Soy Analista de Sistemas egresado del **Instituto Valle Grande** (Cañete, Lima), formado bajo el **método de alternancia** — ciclos de teoría y práctica aplicada donde construí proyectos reales de software desde el primer ciclo.

Me especializo en el **back-end con Java y Spring Boot**, pero tengo experiencia full stack: he trabajado con React, Angular, Flutter y Svelte en distintos proyectos. También he explorado el mundo de la **Web3** y el consumo de **APIs de inteligencia artificial**.

- 🔧 Back-end con Java 17, Spring Boot Reactivo y arquitectura hexagonal
- 🌐 Front-end con React, Angular y Svelte
- 📱 Móvil con Flutter
- 🗄️ Bases de datos: PostgreSQL, Oracle, MySQL, SQL Server
- 🐳 Docker, GitHub, GitLab, despliegue en Render
- ⛓️ Experiencia con blockchain y DApps (Pali Wallet)
- 🤖 Integración de APIs de IA (clasificación de imágenes, generación con prompts)
